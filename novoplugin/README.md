# NOVO PLUGIN CHAMADO BIGTECH

## DESCRIÇÃO
Este plugin adiciona novos provedores de consulta ao sistema, expandindo as capacidades de busca e integração com diferentes fontes de dados. Os arquivos .md estão dividos por categoria/pastas: credito, cadastral e veiculo. neste diretorio, "novoplugin", você encontra os arquivos de configuração e documentação necessários para a instalação e utilização do plugin.

* Casdrastral
 - 320-Contatos Por CEP.md
 - 327 - QUOD CADASTRAL PF.md
 - 424-ValidaID - Localizacao.md
 - 431-Dados de CNH.md
* Credito
 - 36-Busca por Nome+UF.md
 - 39-TeleConfirma.md
 - 41 - PROTESTO SINTÉTICO NACIONAL.md
 - 304 - POSITIVO DEFINE RISCO CNPJ.md
 - POSITIVO ACERTA ESSENCIAL PF.md
* Veículo
 - 411-Crlv-RO.md
 - 412-Crlv RR.md
 - 415-Crlv-SE.md
 - 416-Crlv-SP.md

## INSTRUÇÕES DE IMPLEMENTAÇÃO

* Analise a documentação da pasta "Docs" para verificar como os outros plugins foram criados.
* Adapte os arquivos de configuração conforme necessário para integrar o novo plugin ao sistema existente.
* Certifique-se de testar o plugin em um ambiente de desenvolvimento antes de implementá-lo em produção.
* Crie testes reais para garantir que o plugin funcione conforme o esperado.
